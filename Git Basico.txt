CLONAR REPOSITORIO
git clone url proporcionada

Dirigirse a un directorio 
cd  Help-Desk

Acceder a una rama 
git ls-remote --heads origin boni

Actualizar la rama
git fetch origin

Mostrar todos los directorios existentes
git branch -r

Dirigirse una rama
git checkout boni

Subir modificaciones a la rama 
git pull origin boni

PARA SUBIR CAMBIOS
#Verifica que la rama sea boni usando
git branch
git add.  # Agrega todos los cambios. 
git commit -m "Mensaje de commit"  # Confirma los cambios con un mensaje descriptivo
git push origin boni

PARA TRAER CAMBIOS DEL REPOSITORIO REMOTO
git fetch origin
git pull origin boni
